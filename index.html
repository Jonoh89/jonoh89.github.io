Hi, I'm a JavaScript Developer.

I am currently working as a JavaScript Developer in Newcastle-Upon-Tyne. At work, I am using node server side and marionette.js (backbone.js) along side some html game development using pixi.js. At home, I love to try all the latest tools and technologies. I am using react for my web development, react native for my mobile development and phaser for my game development.

Web

I have worked mostly in web development for the past 4 years. From basic html and css with jQuery, right up to the fully bundled es6+ single page marionette applications. I will save you reading a list of the hundreds of JavaScript tools and frameworks I have used so to see specific technologies used for projects, please see my portfolio. In 2016, I believe in order to be a good developer you don't need to know x, y and z framework, but you must be willing to learn and adapt quickly. Over the last two years on the same project, I have helped move from AMD to commonjs, es5 to transpiled es6, grunt to gulp and phaser to pixi. These changes have added business value by speeding up our process, making our code more readable, modular and faster.

Mobile

I have a keen interest in mobile development. In previous roles, I have been involved in native iOS projects at work and have had a go at home also. More recently, with my specialism in JavaScript and with an interest in sharing code, I have been investigating react native at home.

Server

At Bede Gaming, I work on node servers with most of my work being focused on a slot game web server. I also help with a node API server written using hapi. At home, I have briefly used express, sails and my personal favourite koa. Recently, I have worked mainly with NoSQL databases such as redis and mongodb. However, I have previously worked with relational databases like sql server and I understand the advantages to both approaches. In the past, I have worked with C# MVC ASP.NET and I have investigated other server side frameworks.

I also sometimes blog!

I would like to share my experiences and let you know what I am working on. As I have gained more experience, I find fun in picking up new technologies and believe I can help other people learn whilst creating a reminder for myself :).
